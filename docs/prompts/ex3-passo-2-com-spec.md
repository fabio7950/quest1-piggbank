Implemente o componente `DateRangeFilter` para o dashboard financeiro do piggbank.

Baseie-se nos documentos do repositório:

- `docs/specs/SPEC-filtro-periodo.md` — critérios de aceite, escopo e notas técnicas
- `docs/PRD-filtro-periodo.md` — contexto, objetivos e requisitos de negócio
- `CLAUDE.md` — padrões de projeto, stack e convenções de implementação

O componente deve ser cliente e integrar o filtro de período com o dashboard existente, mantendo o comportamento padrão de “últimos 30 dias” quando nenhum intervalo for selecionado.
